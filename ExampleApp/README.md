# ExampleApp

A simple Swift package demonstrating the OutputTracking library with a network request example.

## What it does
- Searches for books using the Apple iTunes API
- Demonstrates output tracking functionality during network operations
- Includes both an executable and a library target

## Usage

### Run the executable

```bash
swift run
```

### Run the tests

```bash
swift test
```

## Structure
- `NetworkRequest` - Executable that performs book searches
- `NetworkRequestLibrary` - Library demonstrating OutputTracking integration
- `NetworkRequestLibraryTests` - Tests for the library
